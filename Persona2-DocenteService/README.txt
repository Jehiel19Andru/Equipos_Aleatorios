Esta carpeta es para la Persona 2: Microservicio de Docente.

Archivos incluidos:
- server.js (código del microservicio docente)
- package.json (dependencias)
- proto/sala.proto (contrato gRPC)

Ejecución:
1. Abre una terminal aquí.
2. Instala dependencias:
	npm install
3. Ejecuta el microservicio:
	node server.js

Este servicio debe estar corriendo antes que los demás.
