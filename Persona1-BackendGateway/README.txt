Esta carpeta es para la Persona 1: Backend principal/Gateway.
Actualmente, esta carpeta está lista para agregar lógica de gateway si el equipo lo requiere.

Ejecución:
1. Abre una terminal aquí.
2. Instala dependencias (si agregas package.json):
	npm install
3. Ejecuta el servidor (si agregas server.js):
	node server.js

Propósito:
Aquí puedes implementar un gateway o backend principal que orqueste la comunicación entre microservicios.
Incluye el archivo .proto en la carpeta 'proto'.
