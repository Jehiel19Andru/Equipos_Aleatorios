Esta carpeta es para la Persona 4: Frontend.

Archivos incluidos:
- server.js (servidor frontend)
- package.json (dependencias)
- public/alumno.html (interfaz alumno)
- public/docente.html (interfaz docente)
- public/styles.css (estilos)
- proto/sala.proto (contrato gRPC)

Ejecución:
1. Abre una terminal aquí.
2. Instala dependencias:
	npm install
3. Ejecuta el servidor frontend:
	node server.js
4. Accede en el navegador a:
	http://localhost:3000/alumno.html
	http://localhost:3000/docente.html
