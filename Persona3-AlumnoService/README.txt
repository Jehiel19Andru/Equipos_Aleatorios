Esta carpeta es para la Persona 3: Microservicio de Alumno.

Archivos incluidos:
- server.js (código del microservicio alumno)
- package.json (dependencias)
- proto/sala.proto (contrato gRPC)

Ejecución:
1. Abre una terminal aquí.
2. Instala dependencias:
	npm install
3. Ejecuta el microservicio:
	node server.js
